## 2. 구현 기능
- 홈페이지 구현
- Bookmark 앱 : url 저장 및 링크 이동
- Blog 앱 : 블로그 업로드 및 수정, 댓글 달기, 검색 기능
- Photo 앱 : 앨범 생성 및 사진 업로드 기능
- 컨텐츠 편집 기능 (Bookmark, Blog, Photo)
